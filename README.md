# Wars-of-the-Roses
玫瑰战争（又译作蔷薇战争 [5]；英语：Wars of the Roses），或称红白玫瑰战争 [7]，指1455到1485年（一说1487年 [4]）间英国兰开斯特家族和约克家族为争夺王位而断续进行的内战，因兰开斯特家族的族徽为红玫瑰，约克家族的族徽为白玫瑰而得名。 [5]
