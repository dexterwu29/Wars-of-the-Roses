# 测试文档 3

本文件位于 `init/c/3.md`，在 init 子目录下。

一、前提设定（非政治化的叙事视角）

以兰开斯特家族与约克家族为两条平行的家族线，强调家族荣誉、代际传承、婚姻与背叛给个体带来的命运波折。
通过亲情、羁绊、野心、赎罪等情感要素，讲清两大家族在错综复杂的人际关系网中如何彼此牵制、相互影响。
二、兰开斯特家族发展线（以家族成员的成长与困境为核心）

起点与性格底色
家族以勇气、忠诚和家族名誉为核心，代际传承强调“守护土地、传承纹章”的家规。
长辈强调礼仪与荣誉，年轻一代在这种传统中寻找自我定位。
代际冲突与情感纠葛
两位主线人物（虚构化的年轻成员）在婚姻与忠诚之间摇摆，家庭成员之间的信任与背叛成为推动故事的情感驱动力。
家族内的竞争并不全是冷硬的权力博弈，更多体现为对未来的焦虑、对家族安全的渴望，以及对个人自由的追寻。
关键转折（情感与命运的拐点）
某位青年因爱情或对“家族利益”的误解走上错路，导致家庭关系紧张；另一位成员选择牺牲小我以保护家族名誉，留下长期的创伤与和解的契机。
结尾意向
家族的裂痕并未完全修复，但通过共同的记忆与和解的仪式，找到新的共同点与传承方式，为后代埋下希望的种子。
三、约克家族发展线（以冷静、筹谋与情感抉择为驱动）

起点与性格底色
约克家族强调独立、智慧、对公平的追求。家族成员以理性、细致的计划能力著称，善于处理复杂的人际关系。
代际传承与情感抉择
年轻一代在家族使命感与个人理想之间寻找平衡；婚姻与盟约成为个人成长的情感试炼场。
内部有人选择以柔克刚、用关系网化解矛盾；也有成员在压力下坚持个人信念，付出代价。
关键转折（情感与命运的拐点）
一段错综复杂的婚姻安排带来误解与冲突，推动家庭成员重新审视“忠诚”的含义。
通过友谊、信任与宽恕，找到重新团结的契机，开辟新的共同生活路径。
结尾意向
约克家族在经历磨难后，学会以更包容的方式维护家族完整，同时为未来建立新的传承模式。
